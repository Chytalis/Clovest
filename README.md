# Clovest
Clovest adalah sebuah platform berbasis web yang dirancang untuk mengatasi permasalahan limbah tekstil melalui pendekatan Circular Economy. Platform ini bertindak sebagai jembatan antara masyarakat dan "Bank Fashion" untuk memfasilitasi donasi pakaian layak pakai serta pengolahannya menjadi produk kreatif yang bernilai tinggi.
